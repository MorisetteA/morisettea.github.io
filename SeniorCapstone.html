All Possible Futures

About

All Possible Futures is the capstone theme for the 2020 seniors in the Interactive Media Design program. During my junior year in the program, I took part in designing the website to showcase the capstone projects. This year is the first year that the IMD Bothell Capstone website will be a joint website that will archive past capstone projects.

Website

uwb.design

My Role

Web Designer

Tools Used

Wordpress, ELementor, and Adobe XD

Duration

Feburary 2020 - April 2020

Requirements

The seniors came up with a list of some components they wanted to have on the homepage.

  • “What is IMD” Section where we explain the major itself and its goals
  • “What is the Capstone Exhibit” Section. This section should explain our capstone theme (All Possible Futures)
  • Links to project pages / short description or preview of project
  • Link to archive of older capstones (not as relevant as this is the first year for a joint site)
  • Some sort of title banner with logo for the current year’s capstone
  • A footer of some sort that will credit everyone who helped on this project

Inspiration

The seniors wanted to figure out how they should organize the website to showcase all of the capstone projects. So there needed to be an exploration on websites that have the same purpose display multiple academic projects. The final 3 websites are ones that present various student projects from the institutions.

  • UWB IMD Website of 2019
  • Seattle Central Creative Academy
  • School of Art + Art History + Design

User Cases

Originally creating these user cases, the target audience was for those visiting the website and the physical exhibition. However, the capstone exhibition was canceled and the user cases were changed to only focus on how the website itself and who will use it. These cases weren't split into which user will be a priority but rather of the possible audience that might appear.

High Fidelity Wireframe

When creating the wireframes, we were told to focus on the homepage and organize the requirements that were stated earlier.

Reflection

The main takeaway from this project was flexibility. This project happened amidst the COVID-19 outbreak, so there was a major shift in the way we had to work together to complete this website. Working remotely, having virtual meetings, and communicating more often due to separation: these changes were drastic. They forced me to be as flexible as possible with a situation that we couldn't control. I had to adjust my schedule and planning to align with the pace of other groups who contributed to the website. Some problems had to be fixed quickly, but it was imperative that the solutions to these problems still maintained upstanding quality.
